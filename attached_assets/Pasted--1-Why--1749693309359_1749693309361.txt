# 📝 심리상담사 수련생 내담자 경험 플랫폼 기획서

## 1. 프로젝트 개요 (Why)

전국민 마음투자지원사업을 통해 심리상담사 수련생들이 내담자 경험을 쌓는 과정에서, 일부 슈퍼바이저가 추가요금(웃돈)을 요구하는 문제가 발생하고 있음. 

수련생들은 사전 정보가 부족해 상담 진행 중에 예상치 못한 추가비용을 부담하게 되는 사례가 보고됨. 이는 상담 교육과 수련 과정의 공정성과 투명성을 훼손함.

👉 본 플랫폼은 **슈퍼바이저 정보를 사전 투명하게 공개**하고, 수련생들이 **안전하고 공정하게 슈퍼바이저를 선택**할 수 있도록 돕는 것을 목적으로 함.  
👉 전국민 마음투자지원사업의 **본래 취지(공정한 내담자 경험 제공)** 를 지키고, **수련생의 권익을 보호**하는 플랫폼을 구축하고자 함.

### 사용자 그룹
- 슈퍼바이저

- 관리자

## 3. 기능 상세

### 3.1 메인페이지

#### 슈퍼바이저 리스트(카드형 노출)
- 프로필 사진
- 이름 / 소속
- 주요 이력 (1~2줄)
- 상담 분야 (카테고리 태그 표시)
- **내담자 경험 가능 여부 (O/X)**
- **전국민 마음투자사업 참여 가능 여부 (O/X)**
- **전국민 마음투자사업 시 추가요금 여부 (없음/있음+금액)**

#### 클릭 시 → 슈퍼바이저 상세 프로필 페이지로 이동

---

### 3.2 슈퍼바이저 기능

- 회원가입/로그인
- 프로필 등록/수정
    - 프로필 사진
    - 이름
    - 소속
    - 주요 이력
    - 학회 및 자격 (다중선택)
    - 대상별 (다중선택)
    - 고민상황별 (다중선택)
    - 감정과 증상별 (다중선택)
    - 특수 경험별 (다중선택)
    - 상담 가능 지역/방식 (온라인/오프라인/혼합 등)
    - 연락 방법 (이메일/전화/기타)
    - **내담자 경험 가능 여부 (Yes/No)**
    - **전국민 마음투자사업 참여 가능 여부 (Yes/No)**
    - **전국민 마음투자사업 시 추가요금 여부 (없음 / 있음 + 금액 기입 가능)**
    - 프로필 공개/비공개 설정
    - 프로필 URL 제공 (공유 가능)
    - (선택사항) 후기 표시 여부 설정

---

### 3.3 수련생 기능

- 회원가입/로그인
- 슈퍼바이저 검색
    - 카테고리 필터링 (다중선택)
    - 키워드 검색
- 슈퍼바이저 프로필 보기
    - 직접 연락
- **마이페이지**
    - 내 상담내역 기록 기능
        - 작성일 자동 저장
        - 에디터 지원 (텍스트 + 리치텍스트 가능)
        - 본인만 열람 가능 (비공개)
        - 수정/삭제 기능
        - 상담 슈퍼바이저 및 상담일 태그 기능 (선택사항)
- **익명 게시판**
    - 게시글 작성 (익명 닉네임 자동 생성)
    - 댓글 기능
    - 게시글/댓글 신고 기능
    - 신고 시 관리자 검토 후 블라인드 처리 가능

---

### 3.4 관리자 기능

- 회원 관리 (슈퍼바이저/수련생)
- 신고/블랙리스트 관리
- 카테고리 관리 (수정/추가 가능)
- 익명 게시판 모니터링/삭제
- 통계 확인
    - 총 슈퍼바이저 수
    - 수련생 수
    - 등록 상담내역 수
    - 익명 게시판 활동 통계

## 5. 기술적 고려 사항

- 마이페이지 에디터:
    - 모바일 대응 필수
    - 텍스트 저장 시 보안(암호화) 적용
- 익명 게시판:
    - 익명 닉네임 자동 부여
    - 내부 UserID/IP 기록(운영 목적)
- 개인정보 보호:
    - 수련생 상담기록은 외부 노출 금지
    - HTTPS 적용
    - 개인정보처리방침/서비스 이용약관 필수
- MVP 단계에서는 결제 및 일정 예약 기능 미포함
- 상담은 직접 연락 기반으로 진행
